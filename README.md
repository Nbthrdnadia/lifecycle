# lifecycle

A p5.js sketch that reveals a photo sequence with pointillism-style dots.

## How to run

You need a **local web server** so the browser can load images from the `assets` folder reliably.

1. Open a terminal in this project folder (`lifecycle`).
2. Start a simple server, for example:

   ```bash
   python3 -m http.server 8080
   ```

3. In your browser, go to [http://localhost:8080](http://localhost:8080).

The sketch runs automatically; it advances through images on its own timing.

To stop the server, press `Ctrl+C` in the terminal.
